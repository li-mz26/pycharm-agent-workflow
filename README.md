# PyCharm Agent Workflow Plugin

使用大模型 Agent 生成工作流的 PyCharm 插件。

## 功能

- 通过自然语言描述生成工作流
- 集成多种 LLM Provider（OpenAI、Claude、Kimi 等）
- 支持代码生成、重构、测试等场景的工作流
- 可视化工作流编辑器

## 技术栈

- Kotlin / Java
- IntelliJ Platform SDK
- Gradle

## 开发

```bash
./gradlew runIde
```

## 安装

从 JetBrains Marketplace 安装，或下载 release 手动安装。
